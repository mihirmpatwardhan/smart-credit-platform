# 🤝 CredSetu - Smart Credit Platform

CredSetu is a premium, production-ready FinTech web application that leverages AI-driven intelligence to evaluate loan applications, estimate credit limits, and dynamically recommend matched loan products. Built with a modern React frontend and a Node.js/Express/MongoDB backend, the platform features a dynamic administrative control panel, real-time stress testing, automated OCR document scanning, and downloadable audit logs and reports.

---

## 🚀 Key Features

### 🧠 AI-Driven Risk Engine
* **Interactive Risk Assessments:** Utilizes the Google Gemini AI model (`gemini-2.5-flash`) to generate multi-sentence risk summaries, strengths, weaknesses, and final recommendations based on applicant financial metrics.
* **Smart Credit Score Estimator:** Calculates credit profiles for applicants who do not know their formal credit score, using regression-like calculations on income, debt, and employment length.
* **Document OCR Scanning:** Parses and auto-extracts annual income and debt values from bank statements and salary slips using vision-based AI.

### ⚙️ Dynamic Loan Rule Management
* **Database-Driven Rule Engine:** Rules like `minIncome`, `maxDebtToIncomeRatio`, and `baseCreditLimit` are fetched directly from MongoDB, allowing instant system-wide updates without code redeployment.
* **AI Matchmaking & Eligibility Checks:** Automatically matches users to active loan products and estimates their monthly EMIs.
* **Stress-Testing Simulator:** Allows administrators to simulate rule changes across all historical applications to analyze impact before committing to new guidelines.

### 🛡️ Administrative Controls
* **Status & Limit Overrides:** Admins can manually override application statuses (Approved/Pending/Rejected) and adjust assigned credit limits.
* **Security & Audit Logs:** Logs critical actions (e.g., changing rules, overrides) with a timestamp and IP tag.
* **Master PDF Export:** Exports complete application logs to a clean, multi-page PDF table for auditing.

### 📊 Immersive User Dashboard
* **Milestone Progress Tracker:** Visual path-to-approval roadmap generated by AI.
* **Financial Charts:** Interactive debt-to-income and credit score charts using Recharts.
* **Interactive PDF Downloads:** Users can download a beautiful report including an Executive Summary, AI Risk Insight, Roadmap, and Matched Loans.

## ⚙️ Environment Configuration

Ensure you create `.env` files in both `client` and `server` directories:

### Server Environment (`server/.env`)
```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
GEMINI_API_KEY=your_gemini_api_key
SMTP_EMAIL=your_smtp_gmail_address
SMTP_PASSWORD=your_app_password
```

### Client Environment (`client/.env`)
```env
VITE_API_URL=http://localhost:5000/api
```

## 🏃 Getting Started

### 1. Clone & Set Up Directory
```bash
git clone https://github.com/mihirmpatwardhan/smart-credit-platform.git
cd smart-credit-platform
```

### 2. Set Up Backend (Server)
```bash
cd server
npm install
npm run dev
```

### 3. Set Up Frontend (Client)
```bash
cd ../client
npm install
npm run dev
```
